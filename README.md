## 📌 About Project

This project is an audit of **Git**, a distributed version control system created by Linus Torvalds in 2005. Git is widely used in modern software development and follows the GNU General Public License v2 (GPL v2).

The project demonstrates understanding of open-source philosophy, Linux environment, and shell scripting through practical implementation.

---

## 🛠️ Scripts Overview

### 🔹 Script 1 — System Identity Report
Displays system information such as:
- Kernel version
- Distribution name
- Logged-in user
- Uptime and date/time
- Open-source license information

---

### 🔹 Script 2 — FOSS Package Inspector
- Checks whether Git is installed
- Displays version and package details
- Shows configuration files
- Includes open-source philosophy note using case statement

---

### 🔹 Script 3 — Disk and Permission Auditor
- Audits important system directories
- Displays permissions, owner, group, and size
- Checks Git configuration files
- Uses loop and command-line utilities

---

### 🔹 Script 4 — Log File Analyzer
- Reads log files line by line
- Counts occurrences of a keyword (default: error)
- Displays summary and last 5 matching lines
- Accepts user input via command-line arguments

---

### 🔹 Script 5 — Open Source Manifesto Generator
- Takes user input interactively
- Generates a personalised open-source philosophy statement
- Saves output to a text file
